# Проект "РИТМ"  
**Распознавание и интеллектуальный технологический  мониторинг**

## Требования
- Windows 7/8/10/11
- [OpenServer](https://ospanel.io/) (рекомендуется версия 5.4.0 и выше)
- PHP (идёт вместе с OpenServer)
- MySQL (идёт вместе с OpenServer)
- Python 3.10+ (скачать с [python.org](https://www.python.org/))

## Установка и запуск

### 1. Установка OpenServer
1. Скачайте и установите OpenServer.  
2. Запустите OpenServer, в настройках убедитесь, что работают модули PHP и MySQL.

### 2. Копирование проекта
1. Скопируйте папку проекта в директорию:  

### 3. Настройка базы данных
1. Откройте phpMyAdmin: [http://localhost/phpmyadmin](http://localhost/phpmyadmin)  
2. Создайте новую базу данных, например `face_recognition`.  
3. Импортируйте в неё файл базы (например `face_recognitiondb.sql`), который находится в проекте.

### 4. Настройка подключения
Укажите параметры подключения:  
```php
<?php
$host = "localhost";
$user = "root";
$pass = "root";
$db   = "face_recognition";
?>
Если проект использует Python-скрипты, установите нужные библиотеки:
 pip install -r requirements.txt
Запуск проекта:
Запустите OpenServer.
Откройте в браузере:
http://face/
В проекте есть Python-скрипты, их можно запускать отдельно. Сначала:
python recognition_server.py 
А затем: python main.py

Все PHP-файлы должны находиться в папке face.
Python-скрипты могут взаимодействовать с базой или файлами проекта, поэтому убедиться, что пути настроены правильно.
